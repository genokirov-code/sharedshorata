📄 Export & Deploy

Workflow за добавяне на нов епизод:
1. Отиди в "Добави Епизод" → попълни формата → Запази
2. Тук: натисни "Изтегли videos.json"
3. Качи изтегления файл на сървъра (замени стария videos.json)
4. Сайтът автоматично ще покаже новия епизод! ✅

🌐 Hosting Инструкции

Option A — Static hosting (Netlify, Vercel, GitHub Pages):

  1. Изтегли videos.json от тук
  2. Постави го в корена на проекта (до index.html)
  3. Deploy-ни проекта

Option B — FTP/cPanel hosting:

  1. Изтегли videos.json
  2. Качи го чрез FTP в public_html/ или www/
  3. Замени стария файл
